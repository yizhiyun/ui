
Description
-----------
This document describe the RESTful API of how to delete or rename the file from hdfs
-------------
### 1. Request URI: cloudapi/v1/handleHdfsFile/['csvfile', 'mergefile']/{fileName}
### 2. Support Format: JSON
### 3. Request Method: POST
### 4. Response Data:
```
{
	"status": "success"
}
```
