
Description
-----------
This document describe the RESTful API of how to delete the file from hdfs

Upload The Specified CSV/XLS/XLSX File
-------------
### 1. Request URI: cloudapi/v1/deletecsv/{fileName}
### 2. Support Format: JSON
### 3. Request Method: POST
### 4. Response Data:
```
{
	"status": "success"
}
```
